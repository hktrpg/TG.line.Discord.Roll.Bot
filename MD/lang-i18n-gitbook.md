# 語言設定（多語系）

HKTRPG 支援以**群組／伺服器**或**私訊**為單位，設定機器人回覆使用的語言。

預設語言為**正體中文（zh-tw）**。

{% hint style="info" %}
**小知識：** 切換語言只會改變機器人的**回覆文字**。擲骰指令前綴不變，例如 `.cc`、`bothelp`、`/lang`、`.in` 等仍然相同。
{% endhint %}

## 支援語言

| 代碼 | 語言 |
| --- | --- |
| `zh-tw` | 正體中文（預設） |
| `zh-hans` | 简体中文 |
| `en` | English |

{% hint style="warning" %}
請使用 `zh-hans` 切換簡體，**不要**使用 `cn` 或 `zh-cn`（不支援）。
{% endhint %}

## 指令一覽

| 指令 | 說明 |
| --- | --- |
| `.lang` 或 `.lang show` | 顯示目前語言 |
| `.lang list` | 列出支援語言 |
| `.lang help` | 顯示說明 |
| `.lang zh-tw` | 設為正體中文 |
| `.lang zh-hans` | 設為简体中文 |
| `.lang en` | 設為英文 |
| `/lang` | Discord Slash 版（功能相同） |

## 兩種設定範圍

語言設定分成兩種範圍，互不干擾：

### 1. 伺服器／群組（Server）

- **誰可以設：** 需要該群組的**管理員權限**
- **影響範圍：** 此伺服器／群組內，所有人看到的機器人回覆
- **不影響：** 其他伺服器、其他群組、任何人的私訊

範例（在 Discord 伺服器或群組中輸入）：

```text
.lang en
.lang zh-hans
.lang zh-tw
```

或使用 Slash：

```text
/lang
```

### 2. 私訊（DM）

- **誰可以設：** 任何人
- **影響範圍：** **只有你**與 HKTRPG 的私訊回覆
- **不影響：** 任何伺服器／群組的回覆語言

範例（直接私訊 HKTRPG）：

```text
.lang en
.lang zh-hans
.lang zh-tw
```

{% hint style="success" %}
想在團裡用繁中、自己私訊查資料時用英文？可以：群組維持 `zh-tw`，私訊單獨設 `.lang en`。
{% endhint %}

## 實際使用流程

### 把整個 Discord 伺服器改成英文

1. 確認你有該伺服器的管理員權限
2. 在任一頻道輸入：

```text
.lang en
```

3. 之後此伺服器內的說明、錯誤訊息、多數功能回覆會改為英文
4. 要改回繁中：

```text
.lang zh-tw
```

### 只改自己的私訊語言

1. 開啟與 HKTRPG 的私訊
2. 輸入：

```text
.lang zh-hans
```

3. 只有你的私訊會變成简体；群組語言不變

### 查目前是什麼語言

```text
.lang show
```

或：

```text
.lang
```

## 常見問題

### Q：我改了私訊語言，為什麼群組還是繁中？

這是預期行為。私訊語言與群組語言是分開儲存的。

### Q：我在群組輸入 `.lang en`，為什麼說沒有權限？

群組／伺服器語言需要**管理員權限**。一般成員只能在**私訊**設定自己的語言。

### Q：Discord 介面語言會跟著變嗎？

不會。`.lang` 控制的是 **HKTRPG 回覆內容**的語言。\
Discord 客戶端介面、以及 Slash 指令名稱的本地化，仍依 Discord 本身的語言設定。

### Q：指令會不會變成英文版（例如 `.cc` 變成別的）？

不會。指令前綴與骰組指令維持不變，只改回覆文字。

### Q：Line / Telegram / WhatsApp 也可以用嗎？

可以。在對應平台的群組或私訊使用 `.lang` 即可（群組同樣需要管理員權限才能改群組語言）。

### Q：設定後沒有效？

語言設定需要資料庫連線才能儲存。若系統暫時無法連線資料庫，會提示無法儲存；稍後再試即可。
