使用說明
基本功能
  • 支援多種擲骰指令：1d100、2d6、1d20 等
  • 可自訂聊天室名稱
  • 支援深色/淺色主題切換
  • 快速輸入常用指令
  • 可儲存聊天記錄
  • 支援多房間同時使用
  • 即時顯示在線人數
  • 支援自訂暱稱
使用方式
  1. 輸入暱稱（預設為「調查員」+隨機數字，可點擊重新生成按鈕更換）
  2. 點擊「加入聊天室」或直接輸入房間名稱
  3. 在輸入框輸入擲骰指令或聊天內容
  4. 點擊「送出」或按 Enter 鍵發送訊息
擲骰指令格式
  • 基本格式：XdY（X為骰子數量，Y為骰子面數）
  • 範例：
    • 1d100：擲一個100面骰
    • 2d6：擲兩個6面骰
    • 1d20：擲一個20面骰
快速指令功能
  • 點擊「快速指令」按鈕展開常用指令面板
  • 預設提供常用擲骰指令：1d2、1d6、1d8、1d10、1d12、1d20、1d100
  • 可自訂最多5個常用指令
  • 點擊指令即可快速發送
其他功能
  • 點擊「快速輸入」可展開常用指令面板
  • 點擊「儲存」可下載聊天記錄（格式：HKTRPG + 房間名稱）
  • 點擊「說明」可查看此使用說明
  • 點擊「深色模式」可切換主題
  • 點擊「重新生成」可隨機生成新暱稱
注意事項
  • 每間房間上限100條訊息
  • 聊天室名稱不可重複
  • 預設房間「公共房間」不可刪除
  • 暱稱長度限制為50個字元
  • 訊息長度限制為1000個字元
