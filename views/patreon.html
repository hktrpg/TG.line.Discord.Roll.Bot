為什麼加入 HKTRPG Patreon？
• 可提升你的群組/帳號為 VIP，解鎖更高功能上限（關鍵字、角色卡、定時訊息、TRPG 扮演發言等）。
• 依方案可取得 5 / 10 個 VIP 名額，適合個人或團隊管理。
• 高階方案可代托管你命名的骰子機械人（Discord / Line / Telegram）。
• 可加入 Discord 支援群，意見回饋更即時：support.hktrpg.com
使用方式：先到 Patreon 訂閱對應方案，再向管理員取得 Patreon KEY，回到此頁登入即可管理名額。

操作紀錄

    名額分配

    設定每個名額要授予給哪個帳號或伺服器(群組)，儲存後即會同步 VIP 權限。

    延遲說明：儲存後資料庫會立即更新，但機器人端有約 最多 5 分鐘 的快取更新延遲，在此期間於 Discord / Line / Telegram 查詢 VIP 等級（如 .patreon level）可能尚未反映最新狀態，請稍候再試。

    欄位說明：
    • 目標 ID：要授權的使用者 ID 或伺服器(群組) ID（例：Discord User ID / Channel ID）
    • ID 類型：選擇這筆 ID 是 User 還是 Channel
    • 平台：此 ID 所屬平台（Discord / Line / Telegram）
    • 顯示名稱：備註用名稱，方便你辨識對象（可留空）
    • 啓用狀態：啓用中 ＝ 這格立即生效並同步權限；停用中 ＝ 暫停這格權限（設定會保留，之後可再啟用）

    如何取得 ID：
    • 所有平台：可直接在目標對話輸入 .admin id 取得你的用戶ID與群組ID（Discord / Line / Telegram / WhatsApp 皆可）
    • Discord（替代方式）：到「使用者設定 → 進階」開啟「開發者模式」後右鍵複製 ID
    • Line：請先用機器人提供的查詢指令/綁定流程取得對應 ID（不同群組與用戶 ID 不同）
    • Telegram：可透過機器人查詢指令或 @userinfobot 類工具查 User ID / Chat ID
